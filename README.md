# Neural ODE Notebooks

- solver benchmarks on damped oscillator, Van der Pol, and Robertson problems
- learning dynamics with ResNet and Neural ODE style models

- `solver_benchmarks_van_der_pol.ipynb`
- `solver_benchmarks_robertson.ipynb`
- `learning_damped_oscillator.ipynb`
- `learning_damped_oscillator_multiseed.ipynb`
- `learning_van_der_pol.ipynb`
- `learning_van_der_pol_multiseed.ipynb`
- `learning_robertson.ipynb`
